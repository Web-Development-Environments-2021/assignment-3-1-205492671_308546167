get /league/current_fixture

returns a json object with two arrays- array of pre played matches of the current fixture and a array of post played matches of the current fixture

Return type

inline_response_200_1

Example data

Content-Type: application/json
{
  "pre_played_matches" : [ {
    "date" : "2000-01-23",
    "match_id" : "RX23231F",
    "home_team" : "home_team",
    "away_team" : "away_team"
  }, {
    "date" : "2000-01-23",
    "match_id" : "RX23231F",
    "home_team" : "home_team",
    "away_team" : "away_team"
  } ],
  "post_played_matches" : [ null, null ]
}

Produces

  • application/json

Responses
